## Context

`frontend/src/components/PublicationsSection.tsx` already fetches Medium articles dynamically. The section just needs its copy updated to position the writing as AI governance / responsible adoption thought leadership, and to add a prominent link to the Medium profile.

Also: the component uses `process.env.REACT_APP_BACKEND_URL` (CRA convention) — in Vite this is always `undefined`. Fix it to use `import.meta.env.VITE_BACKEND_URL` with the same fallback.

**Translation keys** (update in `useTranslation.ts`):
```
publications.title: 'Thought Leadership'
publications.description: 'Writing on AI governance, responsible adoption, and the limits of AI hype. Practical insights from real enterprise AI deployments.'
```

**Add Medium profile CTA** below the section header (before the articles grid):
```tsx
<a href="https://medium.com/@adrian.c.pop" target="_blank" rel="noopener noreferrer">
  Follow on Medium →
</a>
```
Style consistent with existing link buttons.

**Fix env var** (line 32):
```tsx
// Before:
const BACKEND_URL = process.env.REACT_APP_BACKEND_URL || 'https://medynamic.preview.emergentagent.com';
// After:
const BACKEND_URL = import.meta.env.VITE_BACKEND_URL || 'https://medynamic.preview.emergentagent.com';
```
